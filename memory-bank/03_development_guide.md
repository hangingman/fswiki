## 開発環境のセットアップ

### 前提条件

*   Docker
*   Docker Compose

### セットアップ手順

1.  **依存関係のビルドとアプリケーションの起動:**
    ```bash
    make run
    ```
    これにより、Dockerコンテナがビルドされ、アプリケーションがバックグラウンドで起動します。

## ビルドとテスト

*   **ビルド:**
    ```bash
    make build
    ```
